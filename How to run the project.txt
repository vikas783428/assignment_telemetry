✅ 1. How to run the project (correct order)

Go to your project root:

cd C:\Users\Vikas\Desktop\assignment_telemetry\assignment_telemetry
▶ Step 1: Install dependencies
pip install -r requirements.txt

(if that file is broken, install manually later)

▶ Step 2: Run data generation
py code\generator.py

👉 This creates synthetic telemetry data inside data/

▶ Step 3: Run fault detection
py code\detector.py

👉 Detects anomalies/faults in telemetry

▶ Step 4: Run root cause analysis
py code\rca.py

👉 Finds WHY the fault happened

▶ Step 5: Run visualization
py code\visualize.py

👉 Shows graphs/plots

▶ Step 6 (optional): Evaluation
py code\evaluate.py

👉 Compares predicted faults vs ground truth
